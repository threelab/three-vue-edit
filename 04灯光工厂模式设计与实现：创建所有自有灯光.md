# 灯光工厂模式设计与实现：创建所有自有灯光

## 引言

在3D场景中，灯光决定了视觉体验的质量。Three.js提供了多种灯光类型，但直接使用会导致代码耦合。本文介绍如何通过**工厂模式**和**预设系统**来统一管理灯光的创建与配置。

---

- **Schema驱动**: 所有灯光属性通过LightSchema定义

- **预设系统**: 预定义常用光照组合
- **实时预览**: 添加时支持鼠标跟随预览

## 二、属性定义模块 (LightSchema)

### 2.1 类型定义

```typescript
// 灯光属性类型定义
interface LightPropertySchema {
  name: string;
  label: string;
  type: 'number' | 'color' | 'boolean';
  default: number | string | boolean;
  min?: number;
  max?: number;
  step?: number;
}

// 灯光Schema
interface LightSchema {
  type: string;            // 灯光类型ID
  label: string;           // 显示名称
  icon: string;            // 图标
  properties: LightPropertySchema[];
  previewConfig: {
    position: [number, number, number];
    target?: [number, number, number];
  };
}
```

### 2.2 支持的灯光类型

| 类型 | 图标 | 描述 | 核心属性 |
|------|------|------|----------|
| 环境光 | ◐ | 均匀照亮场景 | color, intensity |
| 方向光 | ⬭ | 平行光源 | color, intensity, castShadow |
| 点光源 | ● | 点发射光源 | color, intensity, distance, decay |
| 聚光灯 | ◆ | 锥形光束 | color, intensity, distance, angle, penumbra, decay |
| 半球光 | ⊕ | 模拟天空/地面 | skyColor, groundColor, intensity |
| 矩形光 | ▭ | 矩形区域 | color, intensity, width, height |

### 2.3 预设系统

## 三、工厂类模块 (LightFactory)

### 3.1 核心结构

```typescript
export class LightFactory {
  private schemas: Map<string, LightSchema> = new Map();
  private creators: Map<string, (props: LightProperties) => THREE.Light> = new Map();
  
  constructor() {
    this.registerDefaultLights();
  }
}
```

### 3.2 核心方法

#### 3.2.1 注册灯光

```typescript
// 注册Schema
registerSchema(schema: LightSchema): void {
  this.schemas.set(schema.type, schema);
}

// 注册创建函数
registerCreator(type: string, creator: CreatorFunction): void {
  this.creators.set(type, creator);
}
```

#### 3.2.2 创建灯光

```typescript
create(type: string, options: CreateLightOptions): LightInstance {
  const schema = this.getSchema(type);
  const creator = this.creators.get(type);
  
  // 创建灯光实例
  const light = creator(properties);
  light.position.set(...position);
  
  // 创建辅助对象
  let helper;
  if (type === 'point') {
    helper = new THREE.PointLightHelper(light, 0.3);
  } else if (type === 'spot') {
    helper = new THREE.SpotLightHelper(light);
  } else if (type === 'directional') {
    helper = new THREE.DirectionalLightHelper(light, 1);
  }
  
  return { type, light, helper };
}
```

#### 3.2.3 应用预设

```typescript
applyPreset(presetName: string): LightInstance[] {
  const preset = this.getPreset(presetName);
  const instances: LightInstance[] = [];
  
  preset.lights.forEach(config => {
    const instance = this.create(config.type, {
      position: config.position,
      properties: config.properties
    });
    instances.push(instance);
  });
  
  return instances;
}
```

### 3.3 内置灯光创建器

```typescript
// 方向光
this.registerCreator('directional', (props) => {
  const light = new THREE.DirectionalLight(
    props.color as string,
    props.intensity as number
  );
  light.castShadow = props.castShadow as boolean;
  return light;
});

// 聚光灯
this.registerCreator('spot', (props) => {
  const light = new THREE.SpotLight(
    props.color as string,
    props.intensity as number,
    props.distance as number,
    props.angle as number,
    props.penumbra as number,
    props.decay as number
  );
  light.castShadow = props.castShadow as boolean;
  return light;
});
```

## 四、UI面板 (LightPanel)

### 4.1 面板结构

```vue
<template>
  <div class="light-panel">
    <!-- 预设模板 -->
    <div class="preset-section">
      <button 
        v-for="preset in presets" 
        :key="preset.name"
        @click="applyPreset(preset.name)"
      >
        {{ preset.label }}
      </button>
    </div>
    
    <!-- 当前灯光类型信息 -->
    <div v-if="currentSchema" class="current-type-section">
      {{ currentSchema.icon }} {{ currentSchema.label }}
    </div>
    
    <!-- 属性面板 -->
    <div class="properties-section">
      <div v-for="prop in currentSchema?.properties">
        <input 
          v-if="prop.type === 'color'" 
          type="color" 
          v-model="currentProperties[prop.name]"
          @change="onPropertyChange"
        />
        <input 
          v-else-if="prop.type === 'number'" 
          type="number" 
          v-model.number="currentProperties[prop.name]"
          @change="onPropertyChange"
        />
      </div>
    </div>
  </div>
</template>
```

## 五、完整使用示例

### 5.1 基础使用

```typescript
import { lightFactory } from './factories/LightFactory';

// 创建单个灯光
const pointLight = lightFactory.create('point', {
  position: [2, 3, 2],
  properties: {
    color: '#ffffff',
    intensity: 1.5,
    distance: 10
  }
});

scene.add(pointLight.light);
if (pointLight.helper) {
  scene.add(pointLight.helper);
}
```

### 5.2 应用预设

```typescript
// 一键应用工作室光照
const instances = lightFactory.applyPreset('studio');

instances.forEach(instance => {
  scene.add(instance.light);
  if (instance.helper) {
    scene.add(instance.helper);
  }
});
```

### 5.3 集成到Vue组件

```vue
<script setup>
const startAddLight = (type: string) => {
  selectedLightType.value = type;
  isAddingLight.value = true;
  createPreviewLight();
};

const handleLightPropertyChange = (properties: LightProperties) => {
  currentLightProperties.value = properties;
  updatePreviewLight(); // 实时更新预览

const handleLightClick = () => {
  sceneManager.addLight({
    type: selectedLightType.value,
    properties: currentLightProperties.value,
    position: [x, y, z]
  });
  isAddingLight.value = false;
  removePreviewLight();
};
</script>
```

## 六、设计优势

### 6.1 扩展性

新增灯光类型只需两步：
1. 在LightSchema添加定义
2. 在LightFactory注册创建器

```typescript
// 新增自定义灯光
const customLightSchema = {
  type: 'my-custom-light',
  label: '我的自定义灯光',
  icon: '⭐',
  properties: [ /* ... */ ],
  previewConfig: { position: [0, 5, 0] }
};

lightFactory.registerSchema(customLightSchema);
lightFactory.registerCreator('my-custom-light', (props) => {
  return new MyCustomLight(/* ... */);
});
```

### 6.2 可维护性

- 灯光创建逻辑集中在Factory
- 属性定义统一管理
- UI与逻辑分离

### 6.3 用户体验

- 鼠标跟随预览
- 属性实时更新
- 预设一键应用
- 辅助对象显示

## 七、与几何体工厂的对比

| 特性 | 几何体工厂 | 灯光工厂 |
|------|-----------|---------|
| Schema | 定义几何体属性 | 定义灯光属性 |
| 创建对象 | Mesh | Light |
| 辅助对象 | 无 | 灯光辅助器 |
| 预设 | 无 | 支持4种光照预设 |
| 预览方式 | 鼠标跟随 | 鼠标跟随 + 属性实时更新 |

## 总结

灯光工厂模式将灯光的创建、配置、管理统一起来，配合预设系统大大简化了用户操作。通过Schema驱动和工厂模式，实现了高度可扩展的架构。

**关键收获：**
1. 工厂模式解耦了对象创建与使用
2. Schema驱动使配置更加声明式
3. 预设系统提升了用户体验
4. 实时预览让交互更直观
