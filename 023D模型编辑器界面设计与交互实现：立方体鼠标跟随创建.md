# 3D模型编辑器界面设计与交互实现：立方体鼠标跟随创建

## 功能概述

本阶段主要完成两个核心功能：
1. **整体界面设计**：模仿Three.js编辑器风格，实现顶部菜单栏、中间三维展示区、右侧属性管理区的布局
2. **立方体鼠标跟随创建**：点击添加按钮后，预览立方体跟随鼠标移动，点击场景固定添加，右键取消创建

## 二、立方体鼠标跟随创建功能

### 2.1 功能流程

```
点击工具栏立方体按钮
        │
        ▼
    进入添加模式
        │
        ▼
    创建预览立方体(半透明)
        │
        ▼
    鼠标移动 → 更新预览位置(射线检测)
        │
        ├── 左键点击 → 固定添加立方体
        ├── 右键点击 → 取消添加
        └── ESC键 → 取消添加
```

### 2.2 核心技术原理

**射线检测（Raycasting）**：从相机发射射线，与地面平面求交点，确定鼠标在3D空间中的位置。

```typescript
// 射线检测核心逻辑
const raycaster = new THREE.Raycaster();
const mouse = new THREE.Vector2();
const gridPlane = new THREE.Plane(new THREE.Vector3(0, 1, 0), 0);

// 计算鼠标位置并更新预览立方体
const updatePreviewCubePosition = (event: MouseEvent) => {
  // 转换为标准化设备坐标
  const rect = container.getBoundingClientRect();
  const mouseX = ((event.clientX - rect.left) / rect.width) * 2 - 1;
  const mouseY = -((event.clientY - rect.top) / rect.height) * 2 + 1;
  
  mouse.set(mouseX, mouseY);
  raycaster.setFromCamera(mouse, camera);
  
  // 计算与地面交点
  const intersectPoint = new THREE.Vector3();
  raycaster.ray.intersectPlane(gridPlane, intersectPoint);
  
  if (intersectPoint) {
    previewCube.position.set(
      Math.round(intersectPoint.x * 10) / 10,
      0.5,
      Math.round(intersectPoint.z * 10) / 10
    );
  }
};
```

### 2.3 预览立方体创建

```typescript
const createPreviewCube = () => {
  const geometry = new THREE.BoxGeometry(1, 1, 1);
  const material = new THREE.MeshPhongMaterial({
    color: 0x40a9ff,
    transparent: true,
    opacity: 0.7,
    side: THREE.DoubleSide
  });
  previewCube = new THREE.Mesh(geometry, material);
  previewCube.name = 'PreviewCube';
  sceneManager.scene.add(previewCube);
};

const removePreviewCube = () => {
  if (previewCube) {
    sceneManager.scene.remove(previewCube);
    previewCube.geometry.dispose();
    (previewCube.material as THREE.Material).dispose();
    previewCube = null;
  }
};
```

### 2.4 点击添加逻辑

```typescript
const handleSceneClick = (event: MouseEvent) => {
  if (!previewCube || !sceneManager || !container) return;
  
  const rect = container.getBoundingClientRect();
  const mouseX = ((event.clientX - rect.left) / rect.width) * 2 - 1;
  const mouseY = -((event.clientY - rect.top) / rect.height) * 2 + 1;
  
  mouse.set(mouseX, mouseY);
  raycaster.setFromCamera(mouse, sceneManager.camera);
  
  const intersectPoint = new THREE.Vector3();
  raycaster.ray.intersectPlane(gridPlane, intersectPoint);
  
  if (intersectPoint) {
    // 创建真实立方体
    const cube = sceneManager.createCube({
      size: 1,
      color: getRandomColor(),
      position: [intersectPoint.x, 0.5, intersectPoint.z]
    });
    cube.userData = { id: objectIdCounter };
    sceneManager.addObject(cube);
    
    // 添加到图层列表
    sceneObjects.push({
      id: objectIdCounter,
      name: `Cube_${objectIdCounter}`,
      type: 'Mesh',
      visible: true,
      position: { x: intersectPoint.x, y: 0.5, z: intersectPoint.z },
      rotation: { x: 0, y: 0, z: 0 },
      scale: { x: 1, y: 1, z: 1 }
    });
    objectIdCounter++;
  }
  
  isAddingCube = false;
  removePreviewCube();
};
```

### 2.5 右键取消功能

```typescript
const handleContextMenu = (event: MouseEvent) => {
  if (isAddingCube) {
    event.preventDefault(); // 阻止默认右键菜单
    isAddingCube = false;
    removePreviewCube();
  }
};
```

## 三、事件监听管理

### 3.1 事件绑定

```typescript
onMounted(() => {
  // 鼠标移动事件 - 更新预览位置
  container.addEventListener('mousemove', handleMouseMove);
  
  // 点击事件 - 添加或选择对象
  container.addEventListener('click', handleClick);
  
  // 右键事件 - 取消添加
  container.addEventListener('contextmenu', handleContextMenu);
  
  // 键盘事件 - ESC取消
  window.addEventListener('keydown', handleKeyDown);
});
```

### 3.2 事件解绑

```typescript
onUnmounted(() => {
  container.removeEventListener('mousemove', handleMouseMove);
  container.removeEventListener('click', handleClick);
  container.removeEventListener('contextmenu', handleContextMenu);
  window.removeEventListener('keydown', handleKeyDown);
  
  sceneManager?.destroy();
  removePreviewCube();
});
```

## 七、总结

本阶段完成了3D模型编辑器的界面布局和核心交互功能：

1. **界面设计**：实现了类似Three.js编辑器的布局风格，包含菜单栏、工具栏、场景区和属性面板

这些功能为后续添加更多几何体类型和编辑功能奠定了坚实基础。
