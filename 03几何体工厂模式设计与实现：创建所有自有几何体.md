# 几何体工厂模式设计与实现：创建所有自有几何体

## 引言

在3D模型编辑器中，几何体的创建是核心功能之一。Three.js提供了丰富的几何体类型，但直接使用这些几何体类会导致代码耦合度高、扩展性差。本文介绍如何使用**工厂模式**设计一个统一的几何体创建框架，支持14种原生几何体的创建。

---

## 一、工厂模式的设计理念

### 1.1 设计原则

工厂模式的核心思想是将对象的创建与使用分离，通过统一的接口创建不同类型的对象。在几何体创建场景中，我们遵循以下原则：

- **单一职责**：每个几何体创建器只负责创建一种几何体
- **开闭原则**：新增几何体类型无需修改现有代码
- **依赖倒置**：依赖抽象而非具体实现

### 1.2 架构层次

## 二、核心类设计

### 2.1 GeometryFactory 类结构

工厂类是整个架构的核心，负责管理几何体类型的注册和创建：

```typescript
export class GeometryFactory {
  private schemas: Map<string, GeometrySchema> = new Map();
  private creators: Map<string, (props: GeometryProperties) => THREE.BufferGeometry> = new Map();
  
  constructor() {
    this.registerDefaultGeometries();
  }
  
  // 注册属性定义
  public registerSchema(schema: GeometrySchema): void;
  
  // 注册创建函数
  public registerCreator(type: string, creator: CreatorFunction): void;
  
  // 创建几何体实例
  public create(type: string, options?: CreateOptions): GeometryInstance;
  
  // 创建预览几何体（半透明）
  public createPreview(type: string, options?: CreateOptions): GeometryInstance;
}
```

### 2.2 GeometrySchema 接口设计

Schema定义了几何体的属性结构，支持动态表单生成：

```typescript
export interface GeometrySchema {
  type: string;                    // 类型标识
  label: string;                   // 显示名称
  icon: string;                    // 图标
  properties: PropertySchema[];    // 属性列表
  previewConfig: {                 // 预览配置
    scale: number;
    position: [number, number, number];
  };
}

export interface PropertySchema {
  name: string;                    // 属性名称
  label: string;                   // 显示标签
  type: 'number' | 'color';        // 属性类型
  default: number | string;        // 默认值
  min?: number;                    // 最小值（number类型）
  max?: number;                    // 最大值（number类型）
  step?: number;                   // 步长（number类型）
}
```

| 2 | capsule | 胶囊 | CapsuleGeometry | 圆柱+半球顶，适合角色碰撞体 |
| 3 | circle | 圆形 | CircleGeometry | 2D圆形平面 |
| 4 | cone | 圆锥 | ConeGeometry | 圆锥体，可调整底面半径和高度 |
| 5 | cylinder | 圆柱 | CylinderGeometry | 圆柱体，支持上下半径不同 |
| 6 | dodecahedron | 十二面体 | DodecahedronGeometry | 12个正五边形面 |
| 7 | icosahedron | 二十面体 | IcosahedronGeometry | 20个三角形面 |
| 8 | octahedron | 八面体 | OctahedronGeometry | 8个三角形面 |

| 10 | ring | 圆环 | RingGeometry | 环形平面（带内孔） |
| 11 | sphere | 球体 | SphereGeometry | 球体，支持经纬度分段 |
| 12 | tetrahedron | 四面体 | TetrahedronGeometry | 4个三角形面 |
| 13 | torus | 圆环体 | TorusGeometry | 甜甜圈形状 |

### 3.2 典型几何体创建器实现

#### 示例1：立方体创建器

```typescript
this.registerCreator('cube', (props) => {
  const width = (props.width as number) || 1;
  const height = (props.height as number) || 1;
  const depth = (props.depth as number) || 1;
  return new THREE.BoxGeometry(width, height, depth);
});
```

#### 示例2：球体创建器

```typescript
this.registerCreator('sphere', (props) => {
  const radius = (props.radius as number) || 0.5;
  const widthSegments = (props.widthSegments as number) || 32;
  const heightSegments = (props.heightSegments as number) || 32;
  return new THREE.SphereGeometry(radius, widthSegments, heightSegments);
});
```

#### 示例3：圆环扭结创建器（参数最复杂）

```typescript
this.registerCreator('torusKnot', (props) => {
  const radius = (props.radius as number) || 0.5;
  const tube = (props.tube as number) || 0.2;
  const radialSegments = (props.radialSegments as number) || 32;
  const tubularSegments = (props.tubularSegments as number) || 16;
  const p = (props.p as number) || 2;  // 扭结参数p
  const q = (props.q as number) || 3;  // 扭结参数q
  return new THREE.TorusKnotGeometry(radius, tube, tubularSegments, radialSegments, p, q);
});
```

## 四、使用示例

### 4.1 创建几何体实例

```typescript
// 获取工厂实例
const factory = new GeometryFactory();

// 创建一个红色立方体
const cubeInstance = factory.create('cube', {
  position: [0, 0.5, 0],
  properties: {
    width: 2,
    height: 1,
    depth: 1,
    color: '#ff0000'
  }
});

// 将几何体添加到场景
scene.add(cubeInstance.mesh);
```

### 4.2 创建预览几何体（半透明效果）

```typescript
// 创建预览几何体（用于鼠标跟随）
const previewInstance = factory.createPreview('sphere', {
  position: [0, 0, 0],
  properties: {
    radius: 0.5,
    color: '#40a9ff'
  }
});

// 预览几何体具有60%透明度
// previewInstance.material.opacity === 0.6
```

### 4.3 动态获取属性Schema

```typescript
// 获取所有已注册的几何体类型
const schemas = factory.getAllSchemas();

// 获取特定类型的属性定义
const sphereSchema = factory.getSchema('sphere');
// 返回：{ type: 'sphere', label: '球体', properties: [...] }
```

## 五、设计优势分析

### 5.1 扩展性

新增几何体类型只需两步：

1. **注册Schema**：定义属性结构
2. **注册Creator**：实现创建逻辑

无需修改现有代码，符合开闭原则。

### 5.2 一致性

所有几何体通过统一接口创建，返回统一的`GeometryInstance`对象：

```typescript
interface GeometryInstance {
  type: string;
  geometry: THREE.BufferGeometry;
  material: THREE.Material;
  mesh: THREE.Mesh;
}
```

### 5.3 资源管理

工厂提供统一的资源释放方法：

```typescript
public dispose(instance: GeometryInstance): void {
  instance.geometry.dispose();
  instance.material.dispose();
}
```

## 六、实际应用场景

### 6.1 场景1：工具栏几何体创建

用户点击工具栏按钮时，通过工厂创建几何体：

```typescript
const handleCreateGeometry = (type: string) => {
  const instance = geometryFactory.create(type);
  scene.add(instance.mesh);
};
```

### 6.2 场景2：属性面板动态渲染

根据Schema自动生成属性表单：

```typescript
const schema = geometryFactory.getSchema(selectedType);
// 根据schema.properties渲染输入框、颜色选择器等
```

### 6.3 场景3：拖拽创建预览

鼠标移动时更新预览几何体位置：

```typescript
const updatePreviewPosition = (position: Vector3) => {
  if (!previewMesh) {
    const instance = geometryFactory.createPreview(selectedType);
    previewMesh = instance.mesh;
    scene.add(previewMesh);
  }
  previewMesh.position.copy(position);
};
```

## 七、总结

通过工厂模式设计，我们实现了：

1. **14种原生几何体**的统一创建管理
2. **Schema驱动**的属性配置系统
3. **预览模式**支持（半透明效果）
4. **资源管理**机制（dispose方法）
5. **高扩展性**（新增类型只需注册）

这种设计使得几何体创建逻辑与UI层解耦，便于后续维护和扩展。
